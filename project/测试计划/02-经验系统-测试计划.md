## 二、测试策略

### 2.1 单元测试

- 覆盖所有经验操作方法
- Mock 数据库操作
- 目标覆盖率 >= 85%

### 2.2 集成测试

- 经验创建 → 查询 → 更新 → 删除完整链路
- 使用真实 SQLite 数据库

### 2.3 MVP 验证

- 验证 MVP 阶段行为（检索返回空、记录静默跳过）
